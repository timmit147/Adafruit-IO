# Adafruit-IO app

Gemaakt door Tim Meeuwsen

## Introductie
Ik heb een app gemaakt waarbij je de LEDstrip kleuren kan aanpassen via de wifi. Met de app kan je kiezen uit een stadaard patroon of een bepaalden kleur.
